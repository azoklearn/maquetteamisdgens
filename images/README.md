# Dossier Images - Partenaires

## Instructions

Placez les logos de vos partenaires dans ce dossier avec les noms suivants :

- `alice-delice.png` - Logo Alice Délice
- `mgel.png` - Logo MGEL
- `helloasso.png` - Logo HelloAsso
- `cultura.png` - Logo Cultura
- `lasermaxx.png` - Logo Lasermaxx

## Format recommandé

- **Format** : PNG avec fond transparent (de préférence) ou JPG
- **Dimensions** : Largeur recommandée de 300-500px
- **Poids** : Optimisez vos images pour un poids < 200 Ko

## Note

Si une image n'est pas trouvée, elle ne s'affichera simplement pas sur le site (grâce à l'attribut `onerror` dans le HTML).
