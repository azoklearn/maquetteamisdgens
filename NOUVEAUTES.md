## ✨ Click Spark Effect - NOUVEAU !

### ✅ Effet de particules au clic

Un effet **subtil et élégant** qui ajoute une touche interactive à chaque clic.

### 🎨 Caractéristiques
- ✅ **6 particules** par clic (léger et non-intrusif)
- ✅ **Taille aléatoire** : 3-8px
- ✅ **Couleurs du site** : Orange, Orange clair, Marron foncé
- ✅ **Animation circulaire** : 0.8s en dispersion
- ✅ **Très performant** : Suppression auto du DOM
- ✅ **Fonctionne partout** : Desktop et mobile

### 🎯 Configuration par défaut
```javascript
Particules : 6
Taille : 3-8px
Couleurs : #f75e2a, #FF8A50, #6D4C41, #5D4037
Vitesse : 30-80px
Durée : 0.8s
```

### 📖 Documentation
Consultez `CLICK_SPARK.md` pour personnaliser l'effet !

## 🍔 Menu Hamburger Full Screen

### ✅ Implémenté sur toutes les pages

Un **menu hamburger moderne et immersif** a été ajouté sur **toutes les résolutions** (desktop, tablette, mobile).

### 🎨 Caractéristiques

#### Design
- ✅ **Full screen** : Occupe tout l'écran à l'ouverture
- ✅ **Dégradé élégant** : Marron foncé → Marron → Orange
- ✅ **Motif subtil** : Texture avec points en arrière-plan
- ✅ **Z-index optimisé** : Menu au-dessus de tout le contenu

#### Animations
- ✅ **Hamburger → X** : Transformation fluide des 3 barres
- ✅ **Apparition progressive** : Chaque lien apparaît avec un délai
- ✅ **Effets hover** : Glissement + glow sur les liens
- ✅ **Transitions 0.5s** : Animations douces et professionnelles

#### Liens
- ✅ **Grande taille** : 2rem (desktop), 1.5rem (mobile)
- ✅ **Uppercase** : Lettres majuscules + espacement 2px
- ✅ **Bouton "Faire un don"** : Style spécial blanc sur orange
- ✅ **10 liens** : Navigation complète

### 📱 Responsive
- Desktop : Liens 2rem, espacement généreux
- Tablette : Liens 1.8rem
- Mobile : Liens 1.5rem, optimisé pour le tactile

### 🎯 Fonctionnalités
- Scroll bloqué quand ouvert (UX)
- Fermeture auto au clic sur lien
- Zone de clic optimisée (40px+)
- Accessible au clavier

### 📖 Documentation
Consultez `MENU_HAMBURGER.md` pour :
- Personnaliser les couleurs
- Modifier les animations
- Ajuster les délais
- Dépannage

## 🚀 Comment tester

### 1. Ouvrir `index.html` dans un navigateur

### 2. Tester le menu hamburger
1. Cliquer sur l'icône hamburger (3 barres)
2. Vérifier que le menu occupe tout l'écran
3. Vérifier les animations des liens
4. Cliquer sur un lien → menu se ferme
5. Tester le hover sur les liens

### 3. Vérifier le crédit noanweb
1. Scroller jusqu'en bas de la page
2. Vérifier que le crédit est visible
3. Cliquer sur les 2 liens pour vérifier qu'ils fonctionnent

### 4. Tester sur différents appareils
- Desktop (> 1200px)
- Tablette (768px - 1200px)
- Mobile (< 768px)

### 5. Tester les animations
1. Scroller sur la page
2. Vérifier que les sections apparaissent avec animations
3. Ouvrir `demo-animations.html` pour voir toutes les animations
